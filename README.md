### font-split
コマンドラインで使用する npm パッケージです。<br />
フォントファイルを GoogleFonts のように120個程のファイルへ分割し、サブセット化します。<br />
GoogleFonts に無いフォントを動的なコンテンツで使用するために制作しました。<br />
<br />
リポジトリ→ https://github.com/stwch/fontsplit <br />
